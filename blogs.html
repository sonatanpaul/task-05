Question-1: What are the different ways to select an element in the DOM?

  1. getElementById: Selects a single element by its unique ID.
  2. getElementsByClassName: Selects multiple elements by their class name. It returns an HTMLCollection.
  3. getElementsByTagName: Selects multiple elements by their tag name like div, p, etc.. It also returns an HTMLCollection.
  4. querySelector: Selects the first element that matches a specified CSS selector.
  5. querySelectorAll: Selects all elements that match a specified CSS selector. It returns a NodeList.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

  1. innerHtml returns all text, including html tags, that is contained by an element.
  2. innerText returns all text contained by an element and all its child elements.
  3. textContents is all text contained by an element and all its children that are for formatting purposes only.

Question-3: What is event delegation in the DOM?

  1. Event Delegation is a pattern based upon the concept of Event Bubbling. It is an event-handling pattern that allows you to handle events at a higher level in the DOM tree other than the level where the event was first received.

Question-4: What is event bubbling in the DOM?

  1. Event Bubbling is a concept in the DOM. It happens when an element receives an event, and that event bubbles up to its parent and ancestor elements in the DOM tree until it gets to the root element.
